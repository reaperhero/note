# service


## 声明服务

1、ng g service services/storag

2、app.module.ts 里面引入创建的服务 并且声明

```
import { StorageService } from './services/storage.service'

providers: [StorageService]
```


3、在用到的组件里面
//引入服务
```
import { StorageService } from '../../services/storage.service';

//初始化

constructor(public storage:StorageService) { 

    let s=this.storage.get();

    console.log(s);
}
```
